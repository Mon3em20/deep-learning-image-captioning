# Deep Learning Image Captioning System

## Overview

This project implements an end-to-end Deep Learning Image Captioning System that automatically generates natural language descriptions for images.

The system combines Computer Vision and Natural Language Processing techniques by extracting visual features from images using a pretrained CNN model (ResNet50) and generating captions through an LSTM-based sequence model.

## Dataset

* MS COCO Dataset
* Image-caption pairs used for training and evaluation

## Project Pipeline

1. Data preprocessing and caption cleaning
2. Vocabulary creation
3. Image feature extraction using ResNet50
4. Sequence preparation for training
5. LSTM-based caption generation
6. Model training and evaluation
7. Caption generation for unseen images
